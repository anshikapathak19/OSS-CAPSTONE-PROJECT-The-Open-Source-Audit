## 📘 Project Description  
This repository contains the technical component of the **Open Source Audit** capstone project.  
It includes **five shell scripts** designed to demonstrate Linux skills and open-source philosophy through system inspection, log analysis, package checks, and file automation.

---

## 🧰 Scripts Overview  

1. **`script1.sh` — (System Identity)**  
   Displays system information including **Kernel version**, **Uptime**, and **OS License**.

2. **`script2.sh` — (Package Inspector)**  
   Checks if **Python3** is installed and prints its version along with a philosophy note.

3. **`script3.sh` — (Disk Auditor)**  
   Uses a loop to report file **permissions** and **sizes** of important system directories.

4. **`script4.sh` — (Log Analyzer)**  
   Searches log files for keywords (like **ERROR**) and counts the number of occurrences.

5. **`script5.sh` — (Manifesto Generator)**  
   An interactive script that generates a **personalized Open Source Manifesto** for the user.
